1. What is the difference between map, filter and forEach find in JavaScript?

The map() method returns a new array, the forEach() method returns undefined. Filter() returns an array containing the element that satisfies the condition, but find() returns the element that satisfies the condition.
In summary, map() and filter() create new arrays, forEach() iterates through the original array and executes a provided function, and find() returns the value of the first element in the array that satisfies a provided testing function.

2. What is the difference between Local storage, Session storage and Cookies?

They all are the ways to store data on the client-side in web browsers, but they vary in terms of their storage capacity, usage etc.
1. LocalStorage: LocalStorage is a persistent client-side storage mechanism that allows you to store key-value pairs in a web browser.
2. SessionStorage: Session storage is a great way to improve the performance of your web applications. It is similar to LocalStorage in that it allows you to store key-value pairs in a web browser, but it is used to store data that needs to be available only during a single browser session. The data is cleared when the user closes the browser or tab.
3. Cookies: Cookies are small text files that are stored on the client-side by the web browser. Cookies are used to store small amounts of data that can be sent back to the server with subsequent requests.

3. What is Event loop in JavaScript? How it works?

The event loop is responsible for taking tasks from a queue and processing them one at a time. It constantly checks the queue for tasks and executes them in a continuous loop, ensuring that the browser or runtime environment stays responsive to user interactions.

Explain Functions

1. quizTimer

The `quiztimer` function handles a timer. The function takes a parameter `dismiss` which determines whether to clear the timer or not. If `dismiss` is truthy (null, undefined, or NaN), the `clearInterval` method is called on the global timer variable, which stops the timer. If `dismiss` is falsy, the function starts the timer by calling the `setInterval` method on the global timer variable. The `setInterval` method sets a callback function that will be executed every 1000 milliseconds.

2. chooseQuiz

The `chooseQuiz` function takes an index and a given answer as parameters. It checks if an element with the same ID as the quiz data at the given index exists in the `answers` array. If it exists, it updates the given answer for that quiz. Otherwise, it adds a new quiz with the given answer to the `answers` array. Finally, it calls a `displayAnswers` function with the updated `answers` array as an argument.
